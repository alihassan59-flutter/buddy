# Buddy App (Frontend Only)

Buddy App is a Flutter-based **freelancing marketplace UI** inspired by platforms like **Fiverr** and **Upwork**. This app allows users to browse freelance services, view freelancer profiles, post job requests, and explore categories — all built using a modern and responsive frontend design.

🛠️ **Note:** This is a **frontend-only project**. No backend or authentication logic is implemented.

---

## 📱 Features (UI Only)

- 🎨 Clean and intuitive user interface
- 🔍 Explore services by category
- 👨‍💼 View freelancer profiles with skills and ratings
- 📝 Create a job posting (UI form only)
- 💬 Message interface (static UI for chats)
- 📂 Bottom navigation for easy access to main sections
- 🌗 Light & Dark Mode support (if applicable)

---

## 🚀 App Flow (Frontend Structure)

1. **Splash / Onboarding Screen**
2. **Login / Register UI** (no real authentication)
3. **Home Screen**
   - Featured freelancers
   - Categories
4. **Service Details Screen**
   - Service description
   - Freelancer info
5. **Profile Screen**
   - Freelancer portfolio
6. **Post a Job Screen**
   - Job posting form UI
7. **Chat Screen**
   - Static conversation layout
